# Think: Recall

<command_purpose>Before revisiting a topic, prove you remember it. Recall thesis, key evidence, and open concerns from memory before receiving new input.</command_purpose>

## Target

## Instructions

1. Load and follow the **recall-first** skill (loaded from workspace skills)
2. Attempt to recall from memory before looking anything up
3. Present what you remember and identify gaps

> - (b) `/think:feynman` — test if you truly understand it
> - (c) Continue with the topic — you've earned fresh input
