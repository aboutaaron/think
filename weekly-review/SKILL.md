# Weekly Review

Step back from the daily grind and ask: what actually happened this week?

## The Five Questions

Ask one at a time. Let each answer breathe before moving on.

1. **What did I ship?** — Concrete outputs: docs written, decisions made, code merged, conversations had, things that moved from "in progress" to "done."

2. **What did I learn?** — New information, changed perspectives, skills practiced. Not "I read an article" — what stuck? What changed how you think?

3. **What would I do differently?** — Moments where you rushed, avoided a hard conversation, over-engineered something, or spent time on the wrong thing. No self-flagellation — just honest calibration.

4. **What should I carry forward?** — Unfinished threads that matter. Conversations to follow up on. Ideas that aren't ready yet but shouldn't be forgotten. Things that need attention next week.

5. **Did anything challenge my House Views?** — If the user maintains House Views, check: did this week's experience support, challenge, or extend any existing positions? Flag for PHV update if needed.

## Output

Write a structured reflection. Keep it short — this should take 10 minutes to produce, not 30.

```markdown
# Week of [date range]

## Shipped
- ...

## Learned
- ...

## Would Do Differently
- ...

## Carry Forward
- ...

## PHV Notes
- [Any House View connections, or "Nothing this week"]
```

## Guidelines

- **Bias toward honesty over impressiveness.** This is a private document, not a standup update.
- **"I didn't do much this week" is a valid and important answer.** Name why — was it recovery, context-switching, blocked, or avoiding something?
- **Connect to previous weeks.** If "carry forward" items keep repeating, that's a signal. Name it.
- **Keep it under 300 words.** If it's longer, you're performing, not reflecting.
